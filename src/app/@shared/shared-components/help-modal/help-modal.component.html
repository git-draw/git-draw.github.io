Help

 Git-Draw tool helps you to draw git branch and commits by using the git commands.

Following commands are available

  • git checkout <branch> - Checkout to another branch
  • git checkout -b <branch> - Create a new branch and checkout
  • git commit -m "message" - Add a commit to the checkout branch
  • git merge <branch> - Merge branch
  • git tag -a "tag" - Add a tag to the current commit
  • help - Display help window
  • developer - Display developer info
